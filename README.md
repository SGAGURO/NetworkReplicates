# NetworkReplicates
이 단원에서는 언리얼 복제 기능과 RPC를 활용하여 전반적인 멀티플레이 기초 로직을 학습한다.
![image](https://github.com/user-attachments/assets/657068de-bdbf-4097-ba7e-9fd643bb3f5d)

### 대표 기능
- 팀 기반의 FPS 멀티 플레이
- 변수와 액터의 복제
- 총기류의 발사와 충돌 처리
- 코스매틱한 기능 추가
- 팀 분배 및 스폰 포인트 이동
- 리스폰 및 점수 부여 시스템 등
